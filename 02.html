Lunatic Rave 2

일본에서 개발된 BMS / PMS 구동기. 루나틱 레이브 2라고 읽으며 약칭은 LR2이다. 2015년 현재 BMS계에서 발광 BMS 인터넷 랭킹 등을 집계할 때 가장 표준적으로 쓰이는 구동기이다. 해상도는 기본적으로 640 x 480의 VGA 해상도를 사용하고 있다.

beatoraja

Java 기반의 오픈 소스 Be-Music Script 구동기이다. 흔히 '비토라자'라고 부른다. 최신 버전은 0.8.5. 자바로 만들어지는 데다 오픈 소스라 여러 플랫폼으로 포팅이 용이하며, macOS에서도 네이티브로 구동된다.

Qwilight

2018년 1월부터 Dev Taehui라는 개발자가 개발 중인 한국산 BMS 구동기다. 개발자가 지속적으로 개발에 신경쓰는 편이라 구동이 안정적이며, 다양한 옵션과 기능을 제공한다.
